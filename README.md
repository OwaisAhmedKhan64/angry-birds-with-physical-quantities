# Angry Birds with Physical Quantities
A game inspired by Angry Birds in which you can see the bird's physical quantities such as range, launch angle, launch velocity etc.

More information will be added later on.

How to run:

- Download the zip file "angry_birds_physical_quantities.zip".
- Extract the zip file.
- Run the executable file i.e., "angry_birds_physical_quantites.exe".

References: 
- The gameplay and physics concept of this project were inspired by Rovio Entertainment’s Angry Birds. 
- The sprites, sounds, and visual elements used in this project are sourced from Rovio Entertainment’s Angry Birds, and are used strictly for educational, non-commercial purposes. 
- Game’s code and resources taken from: https://github.com/estevaofon/angry-birds-python.git.
